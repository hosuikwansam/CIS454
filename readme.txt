Group 18 Electronic Health Record

1.Download XAMPP from https://www.apachefriends.org/download.html
2.Locate the folder containing the XAMPP files.
3.Replace the folders htdocs and mysql with the folders provided.
4.Open the XAMPP control panel that can be found in the XAMPP directory
as xampp-control.exe
5.Click the start buttons for Apache and MySQL.
6.Navigate to localhost on any browser.
7.Enjoy!
